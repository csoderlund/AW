Transcript Table

Top row of buttons

ButtonsDescription
Libs Show the table of library counts for the selected transcript(s).
If no SNPs are not covered by reads, its #Libs will be zero.
Variants Show the table of variants that are found in the selected transcript(s).
Exons Show the table of exons that are found in the selected transcript(s).
Draw Draw the align of transcript exons and variants relative to the parent gene.
Align Display a pop-up a window with the alignment, where '+' is blossum62 >0, 'x' is blossum62<=0, and '-' is gap.
Export... A popup window will display the following:
  1. Table of columns - writes the table to file.
  2. Parental proteins - writes (appends) the Ref and Alt proteins to file.
  3. Alignment of parental protein - writes (appends) the Ref/Alt alignment.
    Warning: if you select many or allow this to run on all rows in a big table, it will take a long time.
Copy Trans.. Copy the transcript name from the selected row so that it can be pasted.
Click with the right mouse button to change it to "Copy identifier..",
which will copy the Identifier into the clipboard for pasting elsewhere.

odRmkRemarks opposite direction replicates or SNPs, as follows:
  1. odSNP Two or more AI (p-value<0.05) SNPs have opposite direction. The 'odSNP:' will be followed by the offending library(s), e.g. odSNP: YfMus.
    To view the SNPs:
    1. Select the transcript followed by "Variant".
    2. Turn on the RefAlt Ratio and Score columns for the offending library, e.g. YfMus.
      You will see that one or more have the opposite direction sign before their ratio,
      and the score will be in the opposite direction from 0.5.
  2. odREP The transcript has one or more libs where the summed SNP replicates are in opposite directions.
    For example, if there are 4 replicates of (90:120, 100:85, 100:70, 125:75), the first would be flagged.
    To see the offending replicates,
    1. Select the transcript, then select "Libs";
    2. From the LibList table, select all libraries followed by "replicate".
    3. The offending replicate(s) will have a value in the "Rep pval" column, all other entries have "--".

Library Group Selection

  • Select at least one option from each row, then select "Apply Selection".
  • Ratio=Ref:Alt, Score=Ref/(Ref+Alt), Pval=p-value

Library Ref:Alt Selection

  • Individually select libraries for Ratio, Score and/or Pvalue.

Manipulating the table

  1. Columns can be moved around by selecting a column heading and moving it.
  2. Columns can be sorted by right click (descending); left click or second right click (ascending).
  3. Various columns can be turned on or off by selecting Columns: Select and setting the desired column appropriately.
  4. When a new table is generated, it uses the last selected column settings. However, the order of columns is not preserved for new tables.
