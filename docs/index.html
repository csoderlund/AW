AW - Allele Workbench
AGCoL | AW Home | Download | Docs | Tour
The Allele Workbench (AW) is for the analysis of allele-specific expression. The pipeline has been tested on Linux, and the Java code (runAW and viewAW) has been tested on Linux and MacOS. See Overview.

This package is no longer being maintained, i.e. new versions of Java or MySQL can break it.
It was last tested in 2019.
