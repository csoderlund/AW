Library Table

Top row of buttons:
  • Select one or more rows followed by "Genes" or "Trans", which will show the table of library counts for all gene(s) or transcript(s).

Column descriptions

The database was built using a configuration file, which defines the "Lib Name" and conditions.
Column nameDescription
LIBidmySQL identifier, used internally
Lib NameThe library abbreviation.
conditionsOne or two columns with the condition names, e.g. Strain and Tissue
#RepsNumber of replicas that were sequenced for the library.
SNP RefCoverage for the reference allele.
SNP AltCoverage for the alternative allele.
SNP AllTotal Coverage.
Read RefNumber of reads assigned to reference; if loaded1,2
Read AltNumber of reads assigned to alternative; if loaded1,2
Read AllTotal number of reads; if loaded1,2
Note that some genes or trans have no Ref/Alt difference
in which case the Ref/Alt counts will be zero.

1These columns are not shown if the corresponding data is not in the database.
2Programs such as eXpress (Roberts A, Pachter L (2013) Nat Methods 10: 71-73.) compute the estimated number of total
reads for the reference and alternative by taking into account the variants.

Manipulating the table

  1. Columns can be moved around by selecting a column heading and moving it.
  2. Columns can be sorted by right click (descending); left click or second right click (ascending).
  3. To change the columns shown, select "Select" (next to the "Columns" label at the bottom of the window).
    Selecting/deselecting a column immediately displays or removes it.
  4. When a new table is generated, it uses the last selected column settings.
