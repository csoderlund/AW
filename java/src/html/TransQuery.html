OD Remark Enter odSNP or odRep (corresponds to column odRmk)
See the Trans table Help for more information, briefly:
  • odSNP: The transcript has one or more SNPs with a significant p-value in the
    opposite direction from the transcript.
  • odRep:The transcript has one or more libraries where the summed SNP replicas
    are in opposite directions. For example, if there are 4 replicas of
    (90:120, 100:85, 100:70, 125:75), the first would be flagged.
Has odRmk Shows any transcript with a OD remark.
This is ignored if a remark is entered into the OD text field.
No odRmk Don't show any transcript with a OD remark.
This is ignored if a remark is entered into the OD text field.
GTK Remark2 Enter ATG, Stop, etc (corresponds to column gtkRmk).
The remarks are in regard to whether there is a start_codon (for brevity, ATG), etc.
Has gtkRmk Shows any transcript with a GTK remark.
This is ignored if a remark is entered into the GTK text field.
No gtkRmk Don't show any transcript with a GTK remark.
This is ignored if a remark is entered into the GTK text field.

SearchDescription
Cov(>=20)Has at least one SNP with at least one library with at least 20 reads.
AI(<0.05)Has at least one SNP with at least one AI library.
#SNP Enter a number to only show transcripts with at least that many SNPs.
Note: the database contains all transcripts, even those without SNPs.
#Coding Enter a number to only show transcripts with at least that many coding SNPs.
#Missense Enter a number to only show transcripts with at least that many missense SNPs.
#Damaged Enter a number to only show transcripts with at least that many damaged SNPs.
Note: A damaged SNP has SIFT=deleterious or High in its Effect column.
Indel Select "Has Indel" for transcripts with one or more indels.
Select "No Indel" for transcripts with no indels.
