SNP Query

Top buttonsDescription
Search Trans If no filters are set, show all variants.
With filters, shows the filtered set of variants.
Limit Set filters that will limit the output to variants that may be of most interest.
ClearClear all filters.
Except for Basic, all filters are applied.
1If an exact match is found, it is displayed. Otherwise, the search will return anything that contains the substring.
Searches are not case-sensitive. Putting a "*" at the end of the string returns all transcripts with that prefix.
2Optional data, may not be in the database.

Basic

rsIDEnter a substring1 or a list of comma separate names.
If the variant is in dbSNP, then the name is the rsID2.
If the variant is new, then it is "SNP" followed by a number or "Indel" followed by a number.

Attributes

EffectListEnter a substring1 or a list of comma separated names.
A variant may have one or more functions as it can be in more than one transcript.
An example function is "missense".
VariantThe type of variant, e.g. select SNP and it will only show SNPs.
CodingThe variant is in a coding exon.
SNPMissense are often the interesting SNPs as they change an amino acid character.
Damaging:
If the annotation is from Ensembl Variant Predictor, damaging indicates SIFT=deleterious.
If the annotation is from snpEFF, the damaging are the ones that are marked as 'high'.
(Note: the database could have been built with a snpEFF file that is augmented with SIFT predictions).
SNP Cov(>=20)SNPs with at least one library with coverage>=20.
SNP AI(<0.05)SNPs with at least one librariy with allele imbalance (AI p<0.05)

Library

This search is not performed unless the left hand box(es) do not contain "empty list".
Select from right box followed by "<", or select all with "<<".
SearchDescription
Coverage Read coverage for the SNP.
Enter either or both limits (<= and >=), e.g. >=20 and <=100
Score Ref/(Ref+Alt).
A value near 0 is Alt>Ref and a value near 1 is Ref>Alt.
Enter either or both limits, e.g. <=0.2 or >=0.8.
AI p-val The p-value from the binomial test (e.g. 0.001).
Select Ref>Alt or Alt>Ref to see the corresponding set.
LibrariesSelecting "All" is very strict, as all libraries must pass the filters.
Selecting "Any" requires one of the selected libraries to pass for the SNP.

Coverage, Score and Pvalue can be simultaneously searched.
