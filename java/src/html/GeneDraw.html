The colored boxes are the exons. The black are non-coding. The rest of the colors
are only meaningful if there are multiple transcripts, as stacked exons will be
the same color if their coordinates are exactly the same, otherwise the color will be different.

The variants are the long vertical lines, where red is an indel (not drawn to scale).
The numbers below them correspond to the numbers in the "Vars" and "Pval" popup windows.
A '*' beside a number indicates that there are one or more libraries that are AI for the SNP.
