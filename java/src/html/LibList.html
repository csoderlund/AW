LibList Table

This table is called 'LibList' as it shows per-library coverage values for a list of genes, transcripts or replicates. It can be displayed from three other tables: gene (Libs), transcripts (lLibs) and libList (replicates).

Top row of buttons:

  • Select one or more rows followed by "replicates", which will show the table of replica coverage values for the selected set.

Manipulating the table

  1. Columns can be moved around by selecting a column heading and moving it.
  2. Columns can be sorted by right click (descending) and left click or second right click (ascending).
  3. To change the columns shown, select "Select" (next to the "Columns" label at the bottom of the window).
    Selecting/deselecting a column immediately displays or removes it.
  4. When a new table is generated, it uses the last selected column settings.
    NOTE: since there are different columns based on whether Lib Reps, Gene Libs, or Trans Libs are shown, the 'last' column settings may not always make sense as they are positional.
